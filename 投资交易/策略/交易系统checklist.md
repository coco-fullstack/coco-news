## 开仓前 Checklist

每次开仓前过一遍这个清单：

### 市场环境
- [ ] 当前是趋势还是震荡？（ADX / 均线结构）
- [ ] BTC 走势如何？（大盘方向）
- [ ] 有没有重大事件要来？（CPI、FOMC、解锁）

### 交易计划
- [ ] 明确了入场理由（不是"感觉要涨"）
- [ ] 设好了止损价
- [ ] 算好了仓位大小（单笔风险 ≤ 2%）
- [ ] 设好了止盈目标（至少 2:1 盈亏比）

### 心态检查
- [ ] 不是在报复性交易
- [ ] 不是 FOMO 追高
- [ ] 今天没有超过日亏损限额
- [ ] 情绪平稳，能接受这笔亏损

> 任何一项没打勾，**不开仓**。

## 持仓中

- [ ] 止损单已挂好（不是"心理止损"）
- [ ] 不频繁看盘干扰判断
- [ ] 到了移动止损条件就执行

## 平仓后

- [ ] 记录到 [[交易日志模板]]
- [ ] 不管盈亏，复盘入场逻辑是否正确
- [ ] 结果不影响下一笔的判断
