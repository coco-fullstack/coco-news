# 常用指令速查

## Obsidian 快捷键

| 操作 | 快捷键 |
|------|--------|
| 命令面板 | `Ctrl+P` |
| 搜索文件 | `Ctrl+O` |
| 搜索内容 | `Ctrl+Shift+F` |
| 新建笔记 | `Ctrl+N` |
| 切换编辑/预览 | `Ctrl+E` |
| 插入链接 | `[[` 然后输入文件名 |
| 插入标签 | `#` 然后输入标签名 |

## Git 同步（命令面板 Ctrl+P）

| 操作 | 命令 |
|------|------|
| 拉取最新 | `Git: Pull` |
| 提交所有修改 | `Git: Commit all changes` |
| 推送到远端 | `Git: Push` |
| 查看修改了哪些文件 | `Git: List changed files` |

## Remotely Save（手机同步）

| 操作 | 命令 |
|------|------|
| 手动同步 | `Ctrl+P` → `Remotely Save: Start Sync` |
| 自动同步 | 设置里已配置，每 5 分钟自动同步 |

## Templater（自动模板）

| 操作 | 步骤 |
|------|------|
| 自动研究币种 | 新建笔记 → 点进去 → `Ctrl+P` → `Templater: Insert Template` → 选 `币种自动研究` → 输入币种名 |
| 新建日记 | 在 `日记/` 新建笔记 → 插入 `日记模板` |

## 命令行 Git（电脑 cmd）

先进入目录：
```
cd "C:\Users\WANG JIHANG\OneDrive - 株式会社クラウディオ\デスクトップ\杂\coco"
```

| 操作 | 命令 |
|------|------|
| 查看状态 | `git status` |
| 拉取最新 | `git pull origin main` |
| 提交并推送 | `git add . && git commit -m "说明" && git push origin main` |
| 解决冲突（以远端为准） | `git checkout --theirs .` → `git add .` → `git commit -m "解决冲突"` |
| 查看远端连接 | `git remote -v` |
